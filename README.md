SSENSE's backend services UI Boilerplate

This repo should work as a starting point for any backend service project requiring a UI part.

## Getting Started

### Development

* run:
``` bash
docker-compose up
```
* wait until you get:
```
ui-xxx_1  | ℹ ｢wdm｣: Compiled successfully.
```
* _start writting some code._

### Technologies used

* React
* Webpack 4
* Docker
* Docker compose
